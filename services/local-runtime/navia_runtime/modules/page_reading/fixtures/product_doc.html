Adapter Contract

The adapter contract is a stable interface between page perception, mindmap generation, renderer output, and agentic loop orchestration.

HighSignalPageContext means a structured high-density page view with filtered blocks, source map reference, digest reference, quality report reference, and readiness status.

SourceRef is defined as a jumpback record containing pageId, contentHash, blockId, blockType, order, textQuote, textHash, fallbackText, and confidence.

Procedure: validate JSON schema, run fixture gates, reject ungrounded digest items, and hand off only pass or explicitly degraded evidence.

Because D and C consume these contracts, candidate extractor output must stay isolated and cannot become the public Navia data model.
