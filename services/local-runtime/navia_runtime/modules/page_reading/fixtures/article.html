Companion Reading Architecture

Navia is a local companion reading assistant that extracts page context and helps readers understand long documents.

Structured Page Facts

The page reading module creates paragraphs, chunks, headings, annotations, and a stable content hash for downstream tools.

For example, a mindmap node can point back to the paragraph and chunk that produced it.

Governed Runtime

Because tool calls are governed, adapters cannot bypass the runtime trace or create artifacts without source references.
