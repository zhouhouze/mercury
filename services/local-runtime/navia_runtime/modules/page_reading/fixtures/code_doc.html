Developer Validation Guide

Developers validate the page perception module with schema checks, fixture tests, and runtime boundary tests.

PYTHONPATH=services/local-runtime python3 -m pytest -q services/local-runtime/navia_runtime/modules/page_reading/tests

The command is a code fact because it verifies the A module contract without invoking D, C, B, MCP, Skill, or external APIs.

If validation fails, the stage returns to planning and implementation must not move forward.
