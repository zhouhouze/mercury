# D Contracts

D consumes and emits existing public V1 contracts:

- `AgentEvent`.
- `ToolResult`.
- `ToolCallRecord`.
- `ArtifactRecord`.

D also consumes V1.2 internal contracts:

- `AgenticLoopContext`.
- `AdapterSpec`.
- `AdapterInvocation`.
- `AdapterResult`.

Contract changes require V1.2-0 review.
