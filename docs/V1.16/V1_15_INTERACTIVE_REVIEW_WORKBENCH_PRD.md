# V1.15 Interactive Review Workbench PRD

Legacy alias: V2.15

## 1. Goal

V1.15 provides an interactive review workbench that turns persisted artifacts into a usable review surface.

## 2. User Story

As a developer, I want one page where I can inspect risks, blockers, evidence, Mermaid diagrams, and export context for follow-up work.

## 3. Requirements

- workbench payload artifact.
- HTML workbench output.
- risk lanes.
- blocker board.
- Mermaid diagrams.
- context export.
- all visible facts trace to persisted artifact IDs.

## 4. Non-Goals

- replacing IDE.
- editing repository files.
- running arbitrary commands from the workbench.
