下一步修复建议

  1. 先把原生 Side Panel 内的按钮操作接入可自动定位的方式，避免用屏幕坐标点击。
  2. 为 Side Panel 验收提供稳定的 test id、可访问自动化入口，或通过原生 Side Panel page target 获取 DOM。
  3. 修复 side panel 窄宽度下快捷按钮横向溢出问题，当前 Mindmap 按钮在原生宽度下可见性不足。
  4. 重新跑侧栏内完整流程：读取页面、Debug、提交上下文、总结、问答、Mindmap、刷新恢复。
  5. 只有上述流程都有“真实网页 + 右侧 Side Panel”截图后，才能声明原生侧边栏 UX 验收通过。
