---

## 1. 阶段目标

将 Chat 页签拆成纯前端渲染模块，消费 D 输出的 SSE 和 ArtifactRecord，展示结构化总结、流式回答、工具状态、Mindmap 和 Debug 信息。

B 完成后，用户仍通过现有页面内 Chat tab 使用功能；B 只负责把 Runtime 事实状态转成 UI presentation，不拥有 AgentCore 状态。

---

```text
apps/chrome-extension/src/modules/chat_renderer/

apps/chrome-extension/src/modules/mindmap_renderer/
```
