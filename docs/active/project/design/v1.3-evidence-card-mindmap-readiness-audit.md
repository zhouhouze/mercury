## 3. 实现范围审计

V1.3 只改变 B Mindmap Renderer 的主视觉和交互呈现：

```text
ArtifactRecord(type=mindmap)
-> B 派生 EvidenceCardViewModel
-> Evidence Card Mindmap 主视图
-> source evidence panel
-> 复用 V1.2 jumpback / fallback
```

不改变：

- A Page Reading 的 perception / digest / source map 输出边界。
- C Mindmap 的服务输出边界。
- D Adapter 的 ToolResult / Artifact / Event / Trace 边界。
- Content Script 的用户触发定位职责。
