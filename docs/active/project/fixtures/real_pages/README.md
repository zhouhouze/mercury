# Real Page Fixtures

These fixtures are derived from real web page structures and shortened for V1.0-0 contract validation.

They are not full page mirrors. They preserve the page type, source URL, title, heading hierarchy, and realistic body content needed for PageContext and schema samples.

| Fixture | Page type | Source |
|---|---|---|
| `article.html` | Article | `https://en.wikipedia.org/wiki/Browser_extension` |
| `docs.html` | Technical documentation | `https://developer.chrome.com/docs/extensions/develop/concepts/content-scripts` |
| `github_readme.html` | GitHub README | `https://github.com/wxt-dev/wxt` |

V1.0-0 uses these fixtures only to close the pre-development real-data requirement. Full content-script extraction is implemented later in V1.0-D.
