Browser extension

A browser extension is a small software module that changes or extends a web browser. It can add interface controls, modify page behavior, or connect the browser to another local or remote service.

Purpose

Common extension uses include productivity tools, ad filtering, password management, page translation, accessibility helpers, and developer utilities.

Architecture

Modern browser extensions usually declare permissions in a manifest and split work across background logic, content scripts, and visible user interface pages.

Security and privacy

Extensions can see sensitive browsing context when granted permission, so modern browsers require explicit permission declarations and review risky capabilities.
