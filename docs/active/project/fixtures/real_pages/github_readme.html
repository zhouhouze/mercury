WXT

WXT is a framework for building browser extensions with modern frontend tooling, TypeScript support, and development workflow helpers.

Features

  • Manifest generation for browser extensions.
  • Entrypoints for background logic, content scripts, side panels, and extension pages.
  • Support for frontend frameworks such as React.

Development workflow

The framework is designed to shorten feedback loops while building extension UI and extension entrypoints.
