Content scripts

Content scripts run in the context of web pages and can inspect or alter the DOM of pages that match an extension's permissions and configuration.

Work in an isolated world

A content script uses an isolated JavaScript environment, which helps separate extension logic from scripts that are loaded by the page itself.

Communicate with the extension

Content scripts use extension messaging to send page information to a service worker, side panel, or another extension page.

Inject user interface

Extensions can add interface elements to the current page, but injected UI should avoid style collisions and should not store core application state in the page.
