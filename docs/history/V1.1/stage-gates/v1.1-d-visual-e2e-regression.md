## 2. 开发计划

- 新增 `chrome-inpage-visual-e2e.mjs`，只负责截图证据，不替代功能 E2E。
- 输出实现截图到 `apps/chrome-extension/e2e/screenshots/current/`。

- 截图覆盖：floating default、floating hover、panel 440 push、panel 50vw push、panel overlay、mobile overlay、runtime offline。
- `artifact-mindmap` 继续后续专项，不纳入 V1.1-D 截图必需项。
- 记录浏览器版本、视口、Runtime 状态、fixture 来源。
- 生成人工视觉审查记录，列出与 Image #1/#2 和 PRD 硬约束的差异。

---

## 3. 验收标准

- Unit tests 通过。
- Typecheck 通过。
- Build 通过。
- 功能 E2E 通过。
- Visual E2E 生成截图。
- 人工视觉审查记录当前截图与 reviewed 基线 / 用户图片 / PRD 硬约束差异。
- PRD A-F 真实 Chrome 手工或自动复验通过。

---

## 4. No-Go

- 只有 mock DOM 截图。
- 只有 Side Panel 截图。
- 没有 Runtime 真实链路。
- 没有截图路径和审查结论。
- 截图缺失任一 required state。
