## 10. 验收命令汇总

V1.1 出门依据来自以下已通过命令：

```bash
node scripts/validate_v1_1_doc_readiness.mjs
pnpm --dir apps/chrome-extension test
pnpm --dir apps/chrome-extension typecheck
pnpm --dir apps/chrome-extension build
NAVIA_E2E_HEADLESS=false pnpm --dir apps/chrome-extension e2e:inpage
pnpm --dir apps/chrome-extension e2e:visual
```

关键约束：

- `e2e:inpage` 与 `e2e:visual` 均使用真实 Local Runtime。
- 验收对象为真实 Chrome MV3 unpacked extension 注入普通网页，不是 Side Panel。
- 真实数据来源为 `docs/navia_v1_project_docs/fixtures/real_pages/article.html`。
- 本机 headless Chromium 无法稳定加载 MV3 unpacked extension，真实验收采用 headed Chromium。
