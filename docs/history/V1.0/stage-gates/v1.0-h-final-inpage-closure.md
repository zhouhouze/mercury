# V1.0-H Stage Gate：网页内交互最终收口

版本：V1.0-H In-page Closure Gate
日期：2026-06-02
阶段状态：计划与审计已闭环；最终自动回归通过；真实 Chrome G/E2 复验待人工确认后可声明 V1 complete

---

## 1. 阶段目标

按 `PRD/窗口交互_PRD.md` 的新口径收口 V1，明确当前可交付能力、限制和开发者上手路径。

V1 complete 声明必须覆盖：

- Chrome unpacked extension 可安装。
- 普通网页内浮动球 / hover 小长条可见。
- 网页内 AI 双轨面板可展开和收起。
- push / overlay / resize / collapse recovery 可复现。
- 当前页面可读取并提交到 Runtime。
- 基础对话、总结、问答和 Mermaid mindmap 可用。
- 最近 session 可恢复。
- Runtime 是 AgentCore 状态源，前端不持有核心状态。

---

## 2. PRD 规格检视

对齐项：

- 前端体验以 `PRD/窗口交互_PRD.md` 为 P0 来源。
- Side Panel 不能作为 V1 complete 的验收替代。
- V1 不引入 V2-V5 能力。

仍不验收：

- Figma 最终视觉稿。
- 语音、桌宠、云同步、知识库、RAG、MCP、Skill、多 Agent、浏览器自动操作、深度研究、PPT。

## 4. 验收标准

自动验收：

- Runtime tests 全绿。
- Chrome extension tests 全绿。
- Typecheck 通过。
- Build 通过。
- `manifest.json` 和 `content-scripts/content.js` 严格 UTF-8 解码通过。

真实 Chrome 手工验收：

- Runtime online。
- 读取真实网页成功。
- summary / QA / mindmap 可用。
- Mermaid SVG 或 source fallback 可见。
- 刷新或重开后 session 恢复。
- 收起后页面布局恢复。

## 5. 审计意见

结论：

```text
Go for final regression and documentation closure.
```

致命问题：无。

重大风险：

| 风险 | 等级 | 闭环 |
|---|---|---|
| 文档仍引用历史 Side Panel complete 口径 | 重大 | 最终 README 和 stage-gate README 必须标注新口径 |

闭环结论：

```text
无未闭环致命或重大规格偏差，可进入最终回归和文档收口。
```
