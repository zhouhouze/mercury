# V1.2 Integration Contract Matrix

版本：V1.2 Integration Contract Matrix
日期：2026-06-04

## 1. 目标

本矩阵约束 Integration Codex 的 wiring。Integration Codex 只连接模块，不重写 A/B/C/D 业务逻辑。

V1.2-E 开始前，Integration Codex 必须逐行复核本矩阵。任一 Producer/Consumer/Contract 不匹配时，停止并打回对应模块或 V1.2-0R，不得为了联通而在 Integration 层重写模块逻辑。

## 2. 合同矩阵

| 边界 | Producer | Consumer | Contract | Transport / Call | Integration 责任 |
|---|---|---|---|---|---|
| Chrome -> Runtime | content script / existing pageContext | Runtime session | PageContext | `POST /v1/page/context` | 保持外部 API 兼容 |
| PageContext -> A | Integration | A Page Reading | PageReadingInput | module function | 构造输入，不改 A 逻辑 |

| A -> D/C | A Page Reading | D Adapter Layer / C Mindmap | HighSignalPageContext / PerceptionDigest / SourceMap / PagePerceptionQualityReport | session.activePage extension / module return | 仅在 A-V1.1-0 公共合同冻结且 quality readiness pass 后消费 |
| D -> CoreProvider | D Adapter Layer | piAgentProvider / MockCoreProvider | CoreTurnInput | `CoreProvider.run_turn` | 选择 provider，不让 Core 直接写 store/SSE/UI |
| CoreProvider -> D | piAgentProvider / MockCoreProvider | D Adapter Layer | CoreTurnResult / AdapterIntent | module return | 映射为 AdapterInvocation / response event |
| D -> Adapter | D Adapter Layer | AdapterRegistry | AdapterInvocation | module call | 确保 governance 前置 |
| D -> C | D internal adapter | C Mindmap | MindmapInput | module call | 传递 structuredPage / IDs |

| Integration -> E2E | all modules | acceptance docs | evidence | test scripts | 记录 PRD 复检 |

## 4. Stop Conditions

Integration must stop and return to module or V1.2-0 if:

- field ownership is unclear.
- module output does not match public API docs.
- a module requires a new external API.
- a module requires new SSE event type.
- a module tries to bypass D governance.
- B needs to directly call A/C/MCP/Skill/API.
- CoreProvider tries to write ArtifactRecord, EventStore, SSE, or UI directly.
- piAgent repository, version, or license cannot be locked before implementation.
- PRD coverage matrix 没有对应 evidence 目标。
- A/B/C/D 任一 stage gate 未通过却开始 E 阶段真实验收。
- A-V1.1 high-signal 输出未完成公共合同冻结却被 D/C 依赖 exact shape。
