## 1. 阶段目标

---

```text

```

如需修改既有 `generate_mindmap` 入口，应由 Integration Codex wiring 或回到 V1.2-0 更新迁移方案。

---

## 3. 开发任务

- [ ] 从 StructuredPageContext 读取 heading tree、paragraph annotations、chunks。
- [ ] 选择 high-importance paragraphs/chunks。
- [ ] 生成 Mermaid source。
- [ ] 执行 Mermaid validation。
- [ ] 最多一次 repair。
- [ ] 生成 MindmapNodeSourceMap。
- [ ] 创建 `ArtifactRecord(type="mindmap", metadata.format="mermaid")`。
- [ ] metadata 包含 validation、nodeSourceMap、paragraphIds、chunkIds。

---

## 4. 验收标准

- [ ] 使用真实 article / docs / github_readme fixture 生成 mindmap。
- [ ] Mermaid source 可通过 validator。
- [ ] repair 次数最多 1 次。
- [ ] artifact 包含 `sourcePageId`、`turnId`、`toolCallId`。
- [ ] 主要节点可追溯到 paragraph/chunk。
- [ ] 无法渲染 Mermaid 时可展示 source fallback。
