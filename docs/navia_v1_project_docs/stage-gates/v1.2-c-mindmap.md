---

## 1. 阶段目标

基于 A 输出的 StructuredPageContext 生成可验证 Mermaid mindmap，并让每个主要节点可追溯到源 paragraph/chunk。

C 完成后，D 可以把它作为 internal adapter 调用；B 可以基于 artifact metadata 渲染 Mermaid 或 source fallback。C 不直接创建 UI，不发 SSE，不读取 Chrome DOM。
